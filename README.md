APFTool - is a fast Admin Panel Search Tool written in Python.


# How to start using
For Linux users:
```
git clone https://github.com/fu11p0w3r/adminfinder
cd adminfinder
pip install -r requirements.txt
chmod +x adminfinder.py
python adminfinder.py
```
For Windows
```
Download
cd adminfinder
pip install -r requirements.txt
python adminfinder.py

```
